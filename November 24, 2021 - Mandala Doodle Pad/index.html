Mandala Doodle Pad

A mandala is a spiritual and ritual symbol in Hinduism and Buddhism, representing the universe. I just think they look neat, so I built this web toy that lets you draw them.

If you're using a device that supports input pressure (like Apple Pencil), press harder to draw thicker lines. Or use a gentle touch for wispy hairlines.

The source code for Mandala Doodle pad is available on GitHub. You can help out with issues here. Parts of this web toy's code are licensed under MIT from Mandala-JS.
